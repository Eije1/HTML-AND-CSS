The CSS Box Model

The CSS box model is a fundamental concept that describes how elements are structured and spaced on a web page. Every element is represented as a rectangular box with content, padding, borders, and margins.

Content-Box (Default)

Width/height set the content size. Padding and border are added to the total dimensions.

Total width: width + padding-left/right + border-left/right

Border-Box

Width/height set the total size including padding and border. Content size is calculated automatically.

Total width: width (includes padding and border)

Spacing with Margin and Padding

Margin creates space outside an element's border, while padding creates space inside the border.
